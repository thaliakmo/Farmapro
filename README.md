# 🏥 Farmapro - Sistema de Gestão Farmacêutica

O **Farmapro** é uma solução Back-end completa para o gerenciamento de operações farmacêuticas. O projeto integra uma API desenvolvida em Java com um banco de dados estruturado para garantir a precisão no controle de estoques, pedidos e receitas.

## 🚀 Diferenciais do Projeto
Este projeto demonstra meu domínio sobre o ciclo de desenvolvimento de software:
* **Modelagem de Dados:** Planejamento via Diagrama Entidade-Relacionamento (DER).
* **Persistência:** Scripts SQL automatizados para criação do banco de dados.
* **Testes de Integração:** Coleção de testes para validação de rotas e lógica de negócio.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Java
* **Banco de Dados:** MySQL / SQL (Script incluso no repositório)
* **Testes de API:** Thunder Client (VS Code)
* **Modelagem:** DER (Diagrama Entidade-Relacionamento)

## 📁 Estrutura do Repositório
* `/farmapro`: Contém a lógica de negócio e as classes em Java.
* `farmapro.sql`: Script completo para subir o banco de dados e as tabelas.
* `DER`: Documentação visual da estrutura do banco de dados.
* `thunder-collection_farmapro.json`: Arquivo para importação de testes de rotas da API.

## 📌 Funcionalidades Principais
1. **Gestão de Estoque:** Controle de entrada e saída de medicamentos com validação de quantidades.
2. **Fluxo de Pedidos:** Processamento ordenado de vendas, garantindo a integridade dos dados.
3. **Registro de Receitas:** Campo dedicado para conformidade com medicamentos controlados.
